# 🚀 BizVaani – Voice-First AI Business Coach for Kirana Stores

🌐 **Live Demo:** https://biz-vaani.vercel.app/

BizVaani is a voice-first AI business assistant designed for Indian kirana (retail) store owners. It enables shop owners to manage sales, get business insights, forecast demand, generate GST invoices, and receive risk alerts using natural voice commands in Hindi, Telugu, and English.

## 🎯 Resume Highlights

- Built a Voice-First AI Business Assistant for Indian MSMEs
- Integrated Speech-to-Text and Text-to-Speech pipelines
- Developed an AI-powered business recommendation system
- Implemented ML demand forecasting using XGBoost
- Created a real-time dashboard with live alerts
- Automated GST invoice generation
- Deployed a production-ready application on Vercel
